# 🤖 Genie AI - Voice Assistant

Genie AI is a Python-based voice controlled assistant.

## Features:
- Voice input & output
- Wikipedia search
- Open apps (Chrome, YouTube, etc.)
- System controls (shutdown, lock, volume)
- WhatsApp messaging automation

## Tech Used:
Python, SpeechRecognition, Pyttsx3, PyAutoGUI
